## OpenVela Platform
1. Create a new folder `openblue` in openvela $(ROOTDIR)/external/
2. clone openblue code into openvela $(ROOTDIR)/external/openblue/
   ```
   git clone git@github.com:Frozen935/openblue.git
   ```
3. copy platform/openvela/ into openvela $(ROOTDIR)/external/openblue/
   ```
   cp -r platform/openvela/* $(ROOTDIR)/external/openblue/*
   ```
